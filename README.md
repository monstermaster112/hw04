# hw04
语音识别
剪映克隆声音：上传含有自己声音的视频，等待识别，录一段10s的音频，确保视频的音频是自己的
目录结构
hw04/
├── asr/                      # 语音识别任务目录
│   ├── .vscode/
│   │   └── settings.json     # VS Code 配置
│   ├── main.py               # 语音识别主程序
│   ├── recorded_audio.wav    # 录制的音频文件
│   ├── report.md             # 语音识别调研报告
│   ├── test_audio.wav        # 测试音频文件
│   └── 语音识别.mp4          # 原始视频文件
├── 文本.txt                   # （空文件）
└── 视频.mp4                  # 另一个视频文件

调研的5种语音识别方案：
- OpenAI Whisper - 通用场景准确率天花板
- Vosk - 极致轻量、低功耗、实时
- Sherpa-ONNX - 速度+精度+部署全优（最终选择）
- FunASR - 中文专业级
- Coqui STT - 开源可训练（已停止维护）
最终选型：Sherpa-ONNX

- 全场景适配
- PC友好，CPU推理极快
- 中文优秀，支持方言
- 部署极简，pip一键安装
- Apache 2.0协议，商用无忧
实现功能
1. 录音功能 ：使用麦克风录制音频并保存为WAV文件
2. 音频识别 ：支持在线识别（Google Speech API）
3. 离线识别方案 ：提供基于Whisper的离线识别方案
4. 文件管理 ：检测并显示当前目录中的音频和视频文件
